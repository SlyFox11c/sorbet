### Color Variations

To activate different sorbet color variations, add these CSS classes to Obsidian's **Settings â†’ Appearance â†’ "Additional CSS classes"** field:

| Variation | CSS Class | Description |
|-----------|-----------|-------------|
| Teal Sorbet (default) | `sorbet-teal` | Soft teal and pink accents |
| Strawberry Sorbet | `sorbet-strawberry` | Warm pink and coral tones |
| Orange-Coral Gradient | `gradient-orange-coral` | Vibrant orange to coral gradient |
| Lemon Sorbet | `sorbet-lemon` | Sunny yellow and peach tones |
| Grape Sorbet | `sorbet-grape` | Rich purple and lavender accents |
